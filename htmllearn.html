HTML (HyperText Markup Language) — Complete Study Notes (Part–1)

1. परिचय (Introduction)

HTML क्या है?

HTML (HyperText Markup Language) एक Markup Language है जिसका उपयोग Web Pages (वेब पेज) बनाने और डिज़ाइन करने के लिए किया जाता है।

Important Note

> HTML कोई Programming Language नहीं है, बल्कि एक Markup Language है।

अन्य Markup Languages

  • XML
  • SGML आदि

2. HTML का उपयोग (Uses of HTML)

HTML का उपयोग मुख्य रूप से:

  • Web Pages बनाने में
  • Website Design करने में
  • Web Content Structure तैयार करने में

आजकल वेबसाइट बनाने के दो तरीके प्रचलित हैं:

(A) Drag & Drop Editors (WYSIWYG)

WYSIWYG = What You See Is What You Get

इन Editors में बिना Coding किए Drag & Drop द्वारा वेबसाइट बनाई जा सकती है।

(B) Manual Coding

इसमें HTML Code लिखकर Web Pages बनाए जाते हैं।

Most Important

> यदि Web Development सीखना है तो HTML सबसे पहला और सबसे आवश्यक विषय है।

4. HTML Tags

HTML की पूरी Coding Tags के अंदर लिखी जाती है।

उदाहरण:

<html>

Tag हमेशा Angle Brackets (< >) के अंदर लिखा जाता है।

Tag की संरचना

< Tag Name >

उदाहरण

<html> <body> <head> <title>

5. Opening Tag और Closing Tag

Opening Tag

<html>

Closing Tag

</html>

अंतर

Closing Tag में Tag Name से पहले / (Slash) लगाया जाता है।

तुलना

Opening Tag Closing Tag
<html> </html>
<body> </body>
<head> </head>
<title> </title>

6. HTML Document का Basic Structure

<html> <head> <title> Hello HTML </title> </head> <body> Welcome To World </body> </html>

7. HTML Structure का कार्य

(1) <html>

  • पूरे HTML Document की शुरुआत करता है।
  • सबसे पहले Open होता है।
  • सबसे अंत में Close होता है।

(2) <head>

Head Section में ऐसी जानकारी लिखी जाती है जो Web Page पर दिखाई नहीं देती।

जैसे:

  • Title
  • Meta Information
  • CSS Links
  • JavaScript Links

(3) <title>

Browser Tab का नाम निर्धारित करता है।

उदाहरण:

<title>Hello HTML</title>

Browser में ऊपर Tab पर यही दिखाई देगा।

(4) <body>

सबसे महत्वपूर्ण Tag।

Web Page पर जो कुछ दिखाई देता है, वह सब Body Tag के अंदर लिखा जाता है।

जैसे:

  • Text
  • Images
  • Tables
  • Links
  • Videos
  • Colors
  • Background
  • Buttons

8. Body Tag का महत्व

Body Tag के अंदर निम्न सभी चीजें लिखी जाती हैं:

  • Text
  • Image
  • Table
  • Heading
  • Paragraph
  • Color
  • Background
  • Formatting
  • Lists
Memory Trick

> जो भी Browser पर दिखाई देगा, वह Body Tag के अंदर होगा।

12. HTML में Formatting कैसे होती है?

HTML Editor में

  • Enter दबाना
  • Space देना
  • Paragraph बनाना

इनका Browser Output पर कोई प्रभाव नहीं पड़ता।
Browser केवल HTML Tags को समझता है।

13. HTML Tags के प्रकार

दो प्रकार के Tags होते हैं।

(1) Container Tag

इनमें Opening और Closing दोनों Tag होते हैं।

उदाहरण

<html></html> <body></body> <head></head> <title></title> <b></b>

(2) Empty Tag

इनमें Closing Tag नहीं होता।

उदाहरण

<br>

तुलना

Container Tag Empty Tag
Opening + Closing केवल Opening
Content रखते हैं Content नहीं रखते
उदाहरण: <body></body> उदाहरण: <br>

14. BR Tag (Line Break)

उपयोग
नई Line में जाने के लिए।

Syntax

<br>

उदाहरण

Hello World <br> Welcome

Output

Hello World Welcome

15. BR Tag की विशेषताएँ

  • Empty Tag है।
  • Closing Tag नहीं होता।
  • एक <br> = एक Line Break

Multiple Line Break

Hello <br> <br> <br> World

Output

Hello


World

16. HTML में Enter क्यों काम नहीं करता?

यदि Notepad में

Hello World

लिख दिया जाए,
तो Browser में Output होगा

Hello World

क्योंकि HTML लगातार Spaces और Enter को Ignore करता है।
नई Line के लिए <br> आवश्यक है।

17. BR Tag कहाँ लगाया जाता है?

जहाँ से नई Line चाहिए,
उस स्थान पर <br> लिखते हैं।

उदाहरण

HTML is used for<br> Creating Web Pages

Output

HTML is used for Creating Web Pages

18. कई Line Break

यदि

<br> <br> <br>

लगाएँ
तो तीन Line Break मिलेंगे।

19. Text Formatting

HTML में Formatting Tags द्वारा की जाती है।
Editor के Formatting Options का Browser पर कोई प्रभाव नहीं पड़ता।

20. Bold Tag

Bold करने के लिए
<b>
का उपयोग किया जाता है।

Syntax

<b> Text </b>

उदाहरण

<b>Hello World</b>

Output

Hello World

21. Bold Tag की विशेषताएँ

  • Container Tag है।
  • Opening और Closing दोनों होते हैं।
  • Opening से Closing तक का पूरा Text Bold हो जाता है।

उदाहरण

<b> HTML stands for HyperText Markup Language </b>

पूरा वाक्य Bold दिखाई देगा।

Important Points (Exam Oriented)

  • HTML = HyperText Markup Language
  • HTML एक Markup Language है।
  • Web Page बनाने की पहली भाषा HTML है।
  • HTML Tags Angle Brackets < > में लिखे जाते हैं।
  • Closing Tag में / लगाया जाता है।
  • <html> सबसे पहले Open और सबसे अंत में Close होता है।
  • <head> में Browser संबंधी जानकारी होती है।
  • <title> Browser Tab का नाम दिखाता है।
  • <body> में पूरा Visible Content लिखा जाता है।
  • HTML File हमेशा .html Extension से Save करें।
  • <br> एक Empty Tag है।
  • <b> एक Container Tag है।
  • केवल Enter दबाने से Browser में नई Line नहीं आती।
  • नई Line के लिए <br> आवश्यक है।

Common Mistakes

  • .html Extension भूल जाना।
  • Closing Tag न लगाना।
  • <br> को Closing Tag देना।
  • <title> को Body में लिखना।
  • Body के बाहर Content लिखना।
  • केवल Enter से Line Break की उम्मीद करना।
